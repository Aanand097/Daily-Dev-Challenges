# Easy Task 1: Form Validation

This solution implements form validation for input fields, including required field checks and email validation.

Files:
- `index.html`
- `styles.css`
- `script.js`

How to test:
- Open `index.html` in a browser.
- Try submitting the form with empty fields or invalid email.
- Verify visual error messages appear and the form submission is blocked until valid.
