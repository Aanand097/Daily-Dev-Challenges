# Day 2 Easy Task 2: REST API

This folder contains a simple Express-based REST API.

## Endpoints

- GET /api/tasks - list all tasks
- POST /api/tasks - create a new task
- PUT /api/tasks/:id - update task title or completion state
- DELETE /api/tasks/:id - delete a task

## Run locally

1. Install dependencies:
   npm install
2. Start the server:
   npm start

The API will be available at http://localhost:3000.
